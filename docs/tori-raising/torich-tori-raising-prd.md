## 1. 배경 및 목적

### 1.1 배경
토리치는 적립식 투자 리마인더/관리 서비스로, 사용자가 매월 정해진 날짜에 투자를 완료하도록 돕는 것이 핵심 가치임. 하지만 투자일이 한 달에 1~2회밖에 없어 **앱 진입 빈도가 낮음**.

토리 탭에 이미 "토리의 하루 한 줄"과 "오늘의 명언"으로 매일 들어올 가벼운 동기는 만들어두었지만, **장기적이고 누적되는 동기**는 부재함.

### 1.2 목적
- **매일 앱에 들어오는 강력한 동기 제공** — 캐릭터 성장 + 꾸미기 욕구
- **투자 완료 행동에 대한 명시적 보상** — 도토리 획득으로 진행도 가시화
- **장기 리텐션 강화** — 1주, 1개월, 3개월, 1년 단위로 도달감 분배

### 1.3 성공 지표 (목표)
- DAU/MAU 비율 향상
- 토리 탭 진입율 향상
- 30일/90일 리텐션 향상
- 투자 완료 체크율 향상

## 2. 핵심 컨셉

### 2.1 한 줄 정의
> "매일 앱에 들어와 도토리를 모으고, 토리를 레벨업시키고, 도토리로 옷을 사서 토리를 꾸미는 캐릭터 육성 시스템"

### 2.2 디자인 원칙

| 원칙 | 설명 |
|---|---|
| 본질 행동을 가장 크게 보상 | 투자 완료가 출석보다 5~10배 보상 큼 |
| 가벼운 출석으로도 진행 가능 | 매일 1개씩이라도 쌓임 |
| 단계별 시각 변화로 도달감 | 레벨 구간마다 토리/칭호 변화 |
| 도토리에 사용처 부여 | 상점 → 꾸미기로 화폐 의미 부여 |
| 강제하지 않음 | 안 해도 본 서비스 이용 가능 |

### 2.3 메타포
- 적립식 투자 = 도토리를 꾸준히 모으기
- 다람쥐는 도토리를 모으는 동물 → 자연스러운 연결
- 모은 도토리로 토리를 꾸민다 = 노력의 가시적 결과

## 3. 사용자 플로우

```
[메인 탭(홈)]
  ↓
  ├─→ “적립식 투자 완료” 버튼 체크(납입 1회/중복방지)
  │     ↓
  │     └─→ 도토리 +10 (피드백: 토스트/모달, 레벨/칭호 변동 가능)
  │
  └─→ (선택) 앱 첫 진입/출석 버튼
        ↓
        └─→ 출석 도토리 +1 (+연속 보너스, 토스트)
  ↓
[토리 탭 진입]
  ↓
  ├─→ 토리 메인: 상점에서 구매한 “배경”으로 화면이 꽉 채워짐
  │
  │   - 좌측 상단: Lv + 칭호
  │   - 우측 상단: 도토리(보유)
  │   - 중앙: 토리(장착 아이템 반영)
  │   - 경험치/다음 레벨 정보
  │
  ├─→ 하단 버튼:
  │     - 출석
  │     - 하루의 명언 보기
  │
  └─→ (보조 진입) 상점/꾸미기에서 아이템(특히 배경) 구매 → 즉시 반영
```

## 4. 기능 명세

### 4.1 토리 성장 시스템

5단계 외형 변화 + 7단계 칭호.

#### 외형 단계

| 단계 | 레벨 | 토리 모습 |
|---|---|---|
| 1 | Lv 1-5 | 아기 토리 (작고 동그란, 큰 눈) |
| 2 | Lv 6-15 | 어린 토리 (살짝 자란) |
| 3 | Lv 16-30 | 청년 토리 (활기찬) |
| 4 | Lv 31-60 | 어른 토리 (꼬리 풍성) |
| 5 | Lv 61+ | 현자 토리 (통통, 부드러운 표정) |

#### 칭호

| 레벨 | 칭호 |
|---|---|
| Lv 1-5 | 🌱 새내기 다람쥐 |
| Lv 6-15 | 🌰 도토리 줍는 다람쥐 |
| Lv 16-30 | 📈 꾸준한 투자러 |
| Lv 31-50 | 💰 도토리 사장님 |
| Lv 51-70 | 🎯 적립의 달인 |
| Lv 71-99 | 🧙 복리의 현자 |
| Lv 100+ | 👑 도토리 황제 |

칭호 표시 형식: `🐿️ "꾸준한 투자러" 토리 · Lv.18`

칭호 변경 시 축하 모달 표시.

#### 레벨업 곡선

| 레벨 구간 | 누적 필요 도토리 | 평균 소요 일수 |
|---|---|---|
| Lv 1 → 5 | 30 | 5~7일 |
| Lv 5 → 10 | 100 | 2주 |
| Lv 10 → 20 | 300 | 1~1.5개월 |
| Lv 20 → 35 | 800 | 3개월 |
| Lv 35 → 60 | 2,000 | 6~9개월 |
| Lv 60+ | 5,000+ | 1년+ |

기준: 매일 출석 + 월 2회 투자 사용자가 **3개월에 Lv 20**, **1년에 Lv 60** 도달.

### 4.2 도토리 경제

#### 획득 규칙

| 행동 | 도토리 | 빈도 |
|---|---|---|
| 일일 출석 (앱 첫 진입 또는 “출석” 버튼) | 1개 | 1일 1회 |
| 토리 탭 진입 | 1개 | 1일 1회(선택) |
| **투자 완료 체크(메인 “적립식 투자 완료” 버튼)** | **10개** | 납입 항목 1회당 / 중복방지 |
| 7일 연속 출석 | +5개 | 7일마다 |
| 30일 연속 출석 | +30개 | 30일마다 |
| 한 달 투자 100% 완료 | +20개 | 월 1회 |
| 명언 공유 | 2개 | 1일 3회 한도 |
| 레벨업 (홀수 레벨) | +5개 | 레벨업 시 |
| 레벨업 (10단위) | 한정 아이템 | Lv 10/20/30... |

메인 “적립식 투자 완료” 버튼에서 발생한 `+10` 보상은 `lastInvestmentCompleteDate`(YYYY-MM-DD)로 중복 방지합니다. (MVP 데모 기준)

#### 누적 vs 보유 분리 (중요)

```
totalAcorns (누적):
  - 모든 획득 도토리 합산
  - 절대 감소하지 않음
  - 레벨 계산의 기준

balance (보유):
  - 현재 사용 가능한 도토리
  - 상점에서 구매 시 감소
  - 새로 획득하면 증가
```

획득 시: 둘 다 +1
구매 시: balance만 -X

### 4.3 도토리 상점

#### 카테고리 구조

| 카테고리 | 아이템 예시 | 가격대 (도토리) | MVP |
|---|---|---|---|
| 🎩 모자 | 비니, 야구모자, 실크햇, 왕관, 베레모 | 50~500 | ✅ |
| 👓 안경 | 동그란 안경, 선글라스, 모노클 | 80~250 | ✅ |
| 👔 의상 | 후드티, 정장, 도복, 우주복, 한복 | 200~800 | ✅ |
| 🎒 소품 | 서류가방, 도토리 컵, 신문, 노트북 | 100~400 | ⚠️ 2차 |
| 🌅 배경 | 사무실, 해변, 도서관, 카페 | 300~1500 | ✅ |
| ✨ 한정판 | 시즌 의상 | 500~2000 | ❌ 3차 |

#### MVP 출시 아이템 최소 라인업
- 모자 5종 + 안경 5종 + 의상 5종 = **총 15아이템**

#### 가격 밸런싱 기준
- 매일 출석 + 월 2회 투자 = 약 30~40 도토리/주
- 저가 아이템: 1주 모음
- 중가 아이템: 2~3주 모음
- 고가 아이템 (배경): 1~2개월 모음

---

### 4.4 꾸미기 시스템

#### 장착 규칙
- 각 카테고리당 1개씩 장착 가능(모자/안경/의상/배경)
- "벗기" 옵션 항상 제공
- 장착한 아이템은 토리 탭 메인 화면에 즉시 반영
- 장착 변경은 자유 (비용 없음)

#### 일러스트 파이프라인
- **레이어 합성 방식** — PNG 투명 배경 레이어로 (베이스 토리) + 모자 + 안경 + 의상 합성
- N×M×K 조합 가능 → 디자이너 작업량 최소화
- 토리 외형 단계별 5종 베이스 + 아이템별 단일 레이어로 충분

### 5.1 (요청 반영: 최신 레이아웃 요약)

- `토리` 탭은 **하단 탭에서 진입 시 화면이 꽉 채워진(full screen) 형태**
- 배경은 **상점에서 구매한 배경이 화면 전체를 채움**
- 토리는 **가운데 배치**(일러스트/미리보기 영역)
- 좌측 상단: **Lv + 칭호**, 우측 상단: **도토리(보유)**
- 중앙/상단 근처: **경험치/진행도 + 다음 레벨까지 도토리**
- 하단 고정 영역: **출석** + **하루의 명언 보기**
- (보조) 하단에서 **상점/꾸미기**로 이동(배경 구매/장착 흐름)

### 7.1 1차 출시 (필수)

| 기능 | 포함 |

| 출석/투자 시 도토리 획득 | ✅ |

## 9. 장기 운영 전략 — 만렙 도달 후

### 9.1 문제 인식
헤비 유저는 6개월~1년 내 만렙(Lv 100+) 도달 가능. 만렙 후 동기 부여 메커니즘 필요.

만렙이 무의미해지면 이탈로 직결됨. **만렙 직전의 수개월간을 위해 다음 보상이 항상 보여야 함.**

### 9.2 단계별 대응

#### 단계 1: 무한 칭호 + 도감 (출시 1개월 내)
- Lv 100 이후로도 칭호 추가 (Lv 150 / 200 / 300...)
- 보유 아이템 도감 — 컬렉션 완성률 표시
- 빈 슬롯이 보이면 채우고 싶어짐

#### 단계 2: 시즌 시스템 (출시 3개월 후)
- 분기별 시즌 도토리 / 한정 아이템
- 본 레벨은 유지, **시즌 진행도는 따로**
- 시즌 종료 시 한정판 아이템 절판 → **희소성 발생**
- 새 시즌마다 "이번 분기 토리 룩" 시즌 미션

#### 단계 3: 사회적 동기 (출시 6개월 후)
- 친구 토리 방문 / 좋아요
- 토리 랭킹 (익명 닉네임)
- 함께 키우기 (모든 친구의 출석이 보너스)

#### 단계 4: 명예 시스템 / Prestige (필요 시)
- Lv 100 도달 시 "환생" 옵션 제공
- 레벨 1로 리셋, 영구 보너스 (도토리 +20% 획득 등)
- 환생 횟수 표시 (별 1개, 별 2개...)
- **선택 사항** — 강제 아님

### 9.3 권장 로드맵

| 시점 | 추가 기능 |
|---|---|
| 출시 시 | 기본 시스템 (Lv 100 칭호까지) |

| 출시 6개월 | 친구 시스템 |
| 출시 1년+ | Prestige 도입 검토 |
